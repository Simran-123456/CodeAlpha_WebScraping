# CodeAlpha - Web Scraping Project

## Project Overview
This project is part of the Data Analytics Internship at CodeAlpha.

The project extracts book titles and prices from a website using Python web scraping techniques.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Features
- Extracts book information from website
- Stores data in CSV format
- Automates data collection process

- web_scraping.py
- scraped_books.csv
